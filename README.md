# rabbit-hole-lab
An evolving collection of interactive infographics about technology, animals, gaming, culture, and other things I find fascinating.

## About the Project

Each experience uses visual storytelling and interaction to make
complex or interesting subjects easier to explore.

## Technologies

- HTML
- CSS
- JavaScript
- SVG and data visualizations

## Run Locally

Clone the repository and open `index.html`, or run it with a local development server.
